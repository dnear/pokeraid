# Pokémon Go Remote Raid Bot

Bot Telegram untuk mengkoordinasi raid Pokémon Go remote.

## Fitur

- ✅ Buat raid baru dengan `/startraid <Pokémon Name>`
- ✅ Tombol interaktif untuk join/leave raid
- ✅ Daftar raid aktif dengan `/list`
- ✅ Lihat raid yang diikuti dengan `/myraids`
- ✅ Pencegahan pesan biasa di group
- ✅ Database SQLite untuk penyimpanan data
